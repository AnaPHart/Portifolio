Sequencias

Sequencias numéricas, na matemática, são definidas como sequencias em que se pode estabelecer uma ler de formação para que se possa saber seus próximos termos. É uma função que cada elemento n corr3sponde a apenas um elemento a. Assim, uma sequência pode tanto ser finita como infinita.

Existem algumas formas de descobrir a lei de formação de uma determinada sequência.

A primeira é a fórmula de recorrência. Existe uma fórmula para o primeiro termo e outra para os próximos. Dessa forma, comparando as duas é possível descobrir a lei de formação, segue o exemplo:

Outro jeito é estabelecer uma formula de na em função do n. Ou seja, se n = 1, a1 = a 4, e assim por diante. É conhecida como termo geral.

E por último, pode-se descobrir pão propriedade de termos. Essa só existe quando é informado na questão a propriedade dos termos. Um exemplo é uma sequência infinita formada pelos números primos.

Progressão aritmética

Parecida com uma sequência, progressão aritmética acontece quando cada termo é a soma do anterior com um número constante, que se chama r. É importante mencionar que só vale a partir do 2 termo. Segue exemplos no slide 1.

As PAs podem ser classificadas em:

Crescentes: os termos são maiores que o anterior;

Constante, o termo é igual ao anterior;

Decrescente: o termo é menor que o anterior;

Uma maneira fácil de descobrir o enésimo termo em uma PA é seguir a fórmula:

an = a1 + (n-1)r

Obs: As vezes é preciso considerar o primeiro termo como a0, então a formula fica an = a0 + (n-1)

O gráfico de uma PA não é uma reta, pois somente os termos que aparecem na sequência fazem parte. Dessa forma, seu gráfico é pontos demarcados, sem ligações. Veja exemplo no slide 2.

Interpolação aritmética é quando se descobre termos do meio em uma sequência. Como o a1 e o an são os extremos, a interpolação acontece quando se obtém a razão para encontrar os termos.

Para que se descubra a soma de todos os temos de uma Pa, sem que se precise somar cada um, existe uma fórmula. Ela é Sn = (a1+a2 )n/2.

Progressão Geométrica

Progressão geométrica acontece quando o termo de uma sequência se dá pelo produto do termo anterior com uma constante chamada q. Importante lembrar que essa fórmula só vale do segundo termo. Segue um exemplo no slide 3.

As PGs podem ser classificadas em:

Crescentes: o termo é sempre maior que o anterior;

Constante: todos os temos são iguais;

Decrescente: o termo é sempre menor que o anterior;

Alternante: o sinal é sempre diferente do anterior;

Estacionaria: apenas o 1 termo é diferente de 0.

A formula do termo geral de uma Pg é dada por an = a1 * q^n-1

Às vezes é preciso que o primeiro termo seja dito como a0, assim a formula fica an = a0 * q^n .

Assim como no gráfico de uma PA serão pontos não ligados, a PG possui os pontos que se fossem ligados formariam uma parábola. Veja exemplo no slide 4.

Para usar a interpolação geométrica em uma PG, se usa a formula n = k + 2, onde k é é o número de termos que ficara entre os extremos. Assim, depois de descoberto o n, se usa a fórmula normal. Segue o exemplo:

Para somar todos os termos de uma PG, usamos a formula Sn = a1 * 1-qn/1-q.

Quando a PG é infinita e queremos domar todos os termos a fórmula é diferente. Assim, a formula fica Sn = a1/1-q onde q é maior que -1 e menor que 1.
